# Arctic Web Map Mapnik Styles

These XML files contain the default styles for our Mapnik server. We currently support the following projections:

* EPSG:3571
* EPSG:3572
* EPSG:3573
* EPSG:3574
* EPSG:3575
* EPSG:3576

The style is based on the default OSM mapnik style.
